# My Personal Website

This repository contains the source code for [**programs.lol**](https://programs.lol), built with **React**, **Vite**, and **TypeScript**.

It also uses the [**Lanyard API**](https://github.com/Phineas/lanyard) to track Discord activity in real-time.



---


To clone this repository:

```bash
git clone https://github.com/realaiden/website.git
cd website
npm install
npm run dev
